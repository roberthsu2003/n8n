# n8n 實戰教學講義

## 關於 n8n

n8n 起初的名稱 nodemation 意指節點自動化，後簡化為 n8n 這種數字縮寫法。n8n 的核心定位是整合帶有 **API 的應用程式和服務**，並強調其 無程式碼（No Code） 或低程式碼的特性。

## n8n 工作流的五個核心邏輯要素

1. **觸發** - 啟動工作流的起點
2. **連接** - 連結不同的服務和應用程式
3. **資料處理** - 轉換和處理資料
4. **測試** - 驗證工作流是否正常運作
5. **啟動** - 執行自動化任務
