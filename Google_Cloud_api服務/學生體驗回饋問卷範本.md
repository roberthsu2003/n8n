# Google_Cloud_api服務
## 學生體驗回饋問卷範本

問卷內有建議欄位,當建議欄位有值時,會自動將發送mail的通知

1. 手動建立Google Form(範例設在roberthsu2004帳號,n8n資料夾內)
2. 表單網址:[學生體驗回饋問卷](https://forms.gle/xK6udQEPLuKSGMHXA)

> 注意事項
> 1. 使用共用讓學員可以複製問卷
> 2. 會自動建立Google Sheet,手動測試表單一次會自動產生Google Sheet
> 3. n8n要取得id,必需使用分享,id會在url中可以找到
> 4. 要將專案inActive,更改為Active,才可以實際測試
> 5. 實際填寫表單,送出時,會自動將資料寫入Google Sheet,在n8n中,Editor是看不到動作,必需要在Execution區域內才可以看到動作
