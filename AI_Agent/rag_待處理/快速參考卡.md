```
RAG 智能文件問答系統
├── 00_使用說明_README.md          ⭐ 完整使用文檔
```

## 🚀 5 分鐘快速啟動

### 步驟 1️⃣: 匯入 Workflow
將 4 個 JSON 檔案匯入 n8n

### 步驟 2️⃣: 設定 Google Gemini
1. 前往: https://aistudio.google.com/app/apikey
2. 建立 API Key
3. 在 n8n 中新增憑證

### 步驟 3️⃣: 上傳文件
開啟 `01_RAG文件索引_本機上傳.json`
→ 取得表單網址 → 上傳檔案

### 步驟 4️⃣: 開始問答
開啟 `03_RAG智能問答_基礎版.json`
→ 取得對話網址 → 提問測試

## ⚙️ 關鍵參數速查

### Text Splitter (文字分割器)
```json
{
  "chunkSize": 800,      // 每個區塊大小
  "chunkOverlap": 200    // 區塊重疊部分
}
```

### Vector Store Tool (檢索工具)
```json
{
  "topK": 4              // 基礎版: 4
                         // 進階版: 5
}
```

### Google Gemini Chat Model
```json
{
  "temperature": 0.3,    // 0.2-0.4 較準確
  "maxOutputTokens": 2048
}
```

## 🔧 快速排除問題

### 問題: 找不到文件
**檢查項目:**
1. Workflow 1 是否執行成功?
2. Embedding 模型是否一致?
   - 都要用 `text-embedding-004`

### 問題: Google Drive 連不上
**檢查項目:**
1. OAuth2 憑證是否有效?
2. 資料夾 ID 是否正確?
3. 是否有存取權限?

### 問題: AI 回答不準確
**調整方法:**
1. 增加 Top K: 4 → 6
2. 降低 Temperature: 0.3 → 0.2
3. 檢查文件品質
