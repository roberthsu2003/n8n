# 截圖說明

## 如何添加 Workflow 截圖

1. 在 n8n 中打開「智能客服聊天機器人」workflow
2. 確保所有節點都在視圖範圍內
3. 使用截圖工具擷取完整的 workflow 畫面
4. 將截圖儲存為 `智能客服聊天機器人.png`
5. 放置於此目錄中

## 建議的截圖內容

截圖應包含：
- ✅ 所有節點（Chat Trigger、AI Agent、OpenAI Chat Model、Simple Memory）
- ✅ 節點之間的連接線
- ✅ Sticky Notes（說明標籤）
- ✅ 清晰可見的節點名稱

## 截圖尺寸建議

- 寬度：1200-1600px
- 高度：800-1200px
- 格式：PNG
- 背景：n8n 預設背景
