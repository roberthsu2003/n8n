## ⚡ 3 步驟快速啟動

### 步驟 1️⃣: 導入 Workflow（2 分鐘）

**方法 A - 從檔案導入**:
```
n8n 介面 → 右上角 ⋮ → Import from File 
→ 選擇「智能客服聊天機器人.json」
```

**方法 B - 從剪貼簿導入**:
```
1. 複製 JSON 檔案內容
2. n8n 介面 → 右上角 ⋮ → Import from Clipboard
3. 貼上並 Import
```

---

### 步驟 2️⃣: 設定 API Key（3 分鐘）

1. **取得 OpenAI API Key**
   - 前往 https://platform.openai.com/api-keys
   - 建立新的 API Key
   - 複製 Key（只會顯示一次！）

2. **設定憑證**
   ```
   點擊「OpenAI Chat Model」節點
   → Credentials → Create New
   → 貼上 API Key
   → Save
   ```

3. **（可選）切換為 Google Gemini**
   - 取得 Google API Key: https://makersuite.google.com/app/apikey
   - 刪除 OpenAI Chat Model 節點
   - 加入 Google Gemini Chat Model 節點
   - 連接到 AI 客服代理節點
   - 設定 Google API Key

---

### 步驟 3️⃣: 測試運行（2 分鐘）

1. **啟動 Workflow**
   ```
   點擊右上角「啟動」按鈕
   ```

2. **開啟聊天介面**
   ```
   點擊「客服聊天」節點
   → 點擊「Test Chat」或複製 Production URL
   ```

3. **測試對話**
   ```
   用戶: 你好
   AI: 您好！很高興為您服務。請問有什麼可以幫助您的嗎？
   
   用戶: 我想了解產品資訊
   AI: 好的！我很樂意為您介紹產品資訊...
   ```

## 🔧 Workflow 架構

```
┌─────────────────┐
│   客服聊天       │ ← 用戶發送訊息
│  (Chat Trigger) │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  AI 客服代理     │ ← AI 處理核心
│   (AI Agent)    │
└─────┬───┬───┬───┘
      │   │   │
      │   │   └─────────────┐
      │   │                 │
      │   ▼                 ▼
      │ ┌──────────┐   ┌──────────┐
      │ │ OpenAI   │   │ 對話記憶  │
      │ │Chat Model│   │ (Memory) │
      │ └──────────┘   └──────────┘
      │      ▲              ▲
      │      │              │
      └──────┴──────────────┘
    連接語言模型      保持對話脈絡
```

## 📊 Workflow 包含節點

| 節點名稱 | 類型 | 版本 | 功能 |
|---------|------|------|------|
| 客服聊天 | Chat Trigger | v1.1 | 接收用戶訊息 |
| AI 客服代理 | AI Agent | v2.2 | AI 處理中心 |
| OpenAI Chat Model | Chat Model | v1 | 語言理解與生成 |
| 對話記憶 | Buffer Window Memory | v1.3 | 記住對話歷史 |

全部使用**最新版本節點** ✅

## 💡 立即自訂

### 修改歡迎訊息
```
點擊「客服聊天」節點
→ Initial Messages
→ 編輯歡迎文字
```

### 調整 AI 個性
```
點擊「AI 客服代理」節點
→ System Message
→ 修改角色定義與個性描述
```

### 調整記憶長度
```
點擊「對話記憶」節點
→ Context Window Length
→ 修改數值（預設 20）
```

## 🆘 遇到問題？

### 常見錯誤速查

| 錯誤訊息 | 可能原因 | 解決方法 |
|---------|---------|---------|
| 401 Unauthorized | API Key 無效 | 重新設定 OpenAI API Key |
| 429 Too Many Requests | API 配額用完 | 檢查 OpenAI 帳戶額度 |
| Workflow not executing | 未啟動 | 點擊右上角「啟動」按鈕 |
| AI 回應慢 | 網路或模型問題 | 切換為 gpt-3.5-turbo |
