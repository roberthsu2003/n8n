# ⚡ 5 分鐘快速開始指南

這是一份超精簡的快速開始指南，讓您在 5 分鐘內啟動並運行 RAG 系統。

---

## 📋 準備工作（2 分鐘）

### 1. 取得 API Keys

#### **HuggingFace API Key**（免費）
1. 前往 https://huggingface.co/settings/tokens
2. 點擊 **New token**
3. 權限選擇 **Read**
4. 複製 Token

#### **OpenRouter API Key**（免費額度）
1. 前往 https://openrouter.ai/
2. 註冊並登入
3. 前往 **Keys** 頁面
4. 建立新 API Key
5. 複製 Key

---

## 🚀 快速設定（3 分鐘）

### **步驟 1：匯入工作流程**（30 秒）

1. 開啟 n8n
2. 點擊 **+** → **Import from File**
3. 選擇 `RAG入門_記憶體儲存.json`
4. 點擊 **Import**

### **步驟 2：設定憑證**（1 分鐘）

#### 2.1 HuggingFace 憑證

1. 點擊任一 `Embeddings HuggingFace Inference` 節點
2. 在 **Credential** 欄位點擊 **Create New**
3. 貼上您的 HuggingFace Token
4. 點擊 **Save**

#### 2.2 OpenRouter 憑證

1. 點擊 `OpenRouter Chat Model` 節點
2. 在 **Credential** 欄位點擊 **Create New**
3. 貼上您的 OpenRouter API Key
4. 點擊 **Save**

### **步驟 3：啟動工作流程**（10 秒）

1. 點擊右上角 **Active** 開關
2. 確認顯示為綠色

### **步驟 4：上傳文件**（30 秒）

1. 點擊 `上傳檔案` 節點
2. 複製 **Production URL**（例如：`https://your-n8n.app/form/abc123`）
3. 在新分頁開啟該網址
4. 上傳一個 PDF 或 CSV 檔案
5. 等待處理完成（顯示「Success」）

### **步驟 5：開始問答**（30 秒）

1. 回到 n8n 工作流程
2. 點擊 `When chat message received` 節點
3. 複製 **Chat URL**（例如：`https://your-n8n.app/chat/def456`）
4. 在新分頁開啟該網址
5. 開始提問！

```

```

## ❌ 遇到問題？

### **問題 1：上傳表單無法開啟**

**檢查**：
- [ ] 工作流程是否已啟動（Active 開關為綠色）
- [ ] Production URL 是否正確複製

### **問題 2：上傳後顯示錯誤**

**檢查**：
- [ ] HuggingFace 憑證是否正確設定
- [ ] 檔案格式是否為 PDF 或 CSV
- [ ] 檔案大小是否小於 10MB

### **問題 3：聊天介面無法開啟**

**檢查**：
- [ ] 是否已先上傳文件
- [ ] Chat URL 是否正確複製
- [ ] 工作流程是否保持啟動狀態

### **問題 4：AI 回答「找不到相關資訊」**

**原因**：
- In-Memory Vector Store 可能已清空（工作流程重啟）
- 需要重新上傳文件

**解決**：
1. 回到上傳表單
2. 重新上傳文件
3. 再次嘗試問答
